in root folder:
	'Preprocess.py' is the python file for data preparation and data preprocessing
	'Preprocessed_Data' folder contains preprocessed data for all experiments
	'DT_Results' folder contains all result files

	in 'Preprocessed_Data' folder:
		DISC1_EXP_A.csv: Preprocessed data set for Experiment 1 using discrete method 1 (binning)
		DISC2_EXP_A.csv: Preprocessed data set for Experiment 1 using discrete method 2 (3-4-5)
		DISC1_EXP_B.csv: Preprocessed data set for Experiment 2 using discrete method 1 (binning)
		DISC2_EXP_B.csv: Preprocessed data set for Experiment 2 using discrete method 2 (3-4-5)
		DISC1_EXP_C.csv: Preprocessed data set for Experiment 3.1 using discrete method 1 (binning)
		DISC2_EXP_C.csv: Preprocessed data set for Experiment 3.1 using discrete method 2 (3-4-5)
		DISC1_EXP_D.csv: Preprocessed data set for Experiment 3.2 using discrete method 1 (binning)
		DISC2_EXP_D.csv: Preprocessed data set for Experiment 3.2 using discrete method 2 (3-4-5)

	in 'DT_Results' folder:
		Discrete_1_A: Result file for Experiment 1 using discrete method 1 (bining)
		Discrete_2_A: Result file for Experiment 1 using discrete method 2 (3-4-5)
		...(similar)
